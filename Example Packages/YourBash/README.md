## Hello!

- This is the manual for the utility YourBash published under lezzetli-mozaik18. For more information, see https://github.com/lezzetli-mozaik18.

- YourBash provides a simple TUI for developers who frequently use bash and have trouble with repetitive commands. It guides and automates to create unique commands with unique purposes that developers can run with the name, simply of all.

- The app may need admin privileges on your device. If you have any doubts, you can always check the source code of the app for predictability.

- Enjoy using YourBash! A desktop app is included for Debian and Ubuntu releases.

- Will not get updates as it is a static app.
